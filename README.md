# Salon Booking System

This project is a complete booking platform for a salon, including a mobile app for customers and a web app for salon admins.

## Project Structure

- `backend/`: Contains the backend server code.
  - `controllers/`: Request handlers for the API routes.
  - `models/`: Database models.
  - `routes/`: API route definitions.
  - `utils/`: Utility functions.
- `frontend/`: Contains the frontend application code.
  - `mobile-app/`: Customer-facing mobile application.
  - `admin-web-app/`: Salon manager web application.
- `docs/`: Contains project documentation.
  - `database-schema.md`: Description of the database schema.
  - `user-flow.md`: User flow diagrams.
  - `notification-workflow.md`: Notification workflow documentation.
  - `slot-blocking-logic.md`: Slot blocking logic documentation.
